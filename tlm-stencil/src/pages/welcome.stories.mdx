### Get started with the MaibornWolff Design System

The MWUI Web Component Libary is based on the MW Design System created and maintained in Figma. The styling of the Web Components is strictly coupled to the Figma Components
using Design Tokens. These Design decisions are versioned and synced between Design and Web, therefore this Storybook represents a synced live state of the
Figma Designs.

There are several ways to include the MWUI components in a project. Build with Stenciljs, all components can be used as **Web Components**. Framwork-specific integrations make it easier to use the Web Components.

### Web Components

Integrate Web Components in any framework or none at all:

1. Install npm package

```bash
npm i tlm-stencil
```

2. Include the following in the `main.js` (e.g. Angular, Vue) or `index.js` (e.g. React):

```javascript
import { defineCustomElements } from 'tlm-stencil/loader';
defineCustomElements(window);
```

### React

1. Install npm package

```bash
npm i tlm-react
```

2. Import and use as a React-like component.

```javascript
import { MWButton } from 'tlm-react';
```

### Vue

1. Install npm package

```bash
npm i tlm-vue
```

2. Import the library plugin in the `main.js` file:

```javascript
import { MWComponents } from 'tlm-vue';
createApp(App).use(MWComponents).mount('#app');
```

3. You can then use your components within vue:

```javascript
<template>
  <mw-button test-id="test-id" label="Press here" />
</template>

<script>
import { MWButton } from 'tlm-vue';

export default {
  name: 'HelloWorld',
  components: {
    MWButton
  }
}
```

> **_NOTE:_** for known issues, see [MWUI Vue Integration](https://github.com/MaibornWolff/tlm-components/tree/main/tlm-vue#known-issues).

### Angular

To enable support for web components in Angular, add the following to the `@NgModule` in `app.module.ts`:

```TypeScript
schemas: [CUSTOM_ELEMENTS_SCHEMA];
```

> **_NOTE:_** The Angular integration is still work in progress
