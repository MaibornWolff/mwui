## Getting Started

> _This project is part of a research project and not an official library._

The styling of the Web Components is strictly coupled to the Figma Components using Design Tokens. These Design decisions are versioned and synced between Design and Web,
therefore this Storybook represents a **synchronized state** of the Figma Designs.

There are several ways to include the MWUI components in a project. Build with Stenciljs, all components can be used as Web Components.
Framwork-specific integrations make it easier to use the Web Components but are not mandatory to use.

</section>

## Web Components

Integrate Web Components in any framework or none at all:

1. Install npm package

<Source language="bash" dark code={`npm i @maibornwolff/mwui-stencil`} />

2. Include the following in the `main.js` (e.g. Angular, Vue) or `index.js` (e.g. React):

<Source
  dark
  language="tsx"
  code={`import { defineCustomElements } from "@maibornwolff/mwui-stencil/loader";
defineCustomElements(window);`}
/>

## Integrations

Framework integrations are optional to use as Web Components run in any browser without specific frameworks or libraries. The provided integrations can help using the Web Components in a more "react, vue or angular" way.

### React

1. Install npm package

<Source
  dark
  language="bash"
  code={`npm i @maibornwolff/mwui-react
npm i @maibornwolff/mwui-stencil`}
/>

2. Import and use as a React-like component.

<Source
  dark
  language="tsx"
  code={`import { MWButton } from "@maibornwolff/mwui-react";
`}
/>

3. Import global styles

<Source
  dark
  language="bash"
  code={`import '@maibornwolff/mwui-stencil/dist/mwui-stencil/mwui-stencil.css';
`}
/>

### Vue

1. Install npm package

<Source
  dark
  language="bash"
  code={`npm i @maibornwolff/mwui-vue
npm i @maibornwolff/mwui-stencil
`}
/>

2. Import the library plugin and global styles in the `main.js` file:

<Source
  dark
  language="tsx"
  code={`import { MwuiComponents } from "@maibornwolff/mwui-vue";
import "@maibornwolff/mwui-stencil/dist/mwui-stencil/mwui-stencil.css";

createApp(App).use(MwuiComponents).mount("#app");
`}
/>

3. You can then use your components within vue:

<Source
  language="tsx"
  dark
  code={`<template>
  <mw-button test-id="test-id" label="Press here" />
</template>

<script>
import { MwButton } from '@maibornwolff/mwui-vue';

export default {
  name: 'HelloWorld',
  components: {
    MwButton
  }
}
`}
/>

> **_IMPORTANT:_** If using `vue 3` and `vite`, [setup your vite.config accordingly](https://stenciljs.com/docs/vue#creating-a-consumer-vue-app).

> **_NOTE:_** for known issues, see [MWUI Vue Integration](https://github.com/MaibornWolff/mwui/tree/main/mwui-vue#known-issues).

### Angular

To enable support for web components in Angular, add the following to the `@NgModule` in `app.module.ts`:

<Source
  dark
  code={`schemas: [CUSTOM_ELEMENTS_SCHEMA];
`}
/>

> **_NOTE:_** The Angular integration is still work in progress
