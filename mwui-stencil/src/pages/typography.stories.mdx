### Extended Typography using injected CSS classes

The following CSS classes can be used independently from the document's semantics to apply styling based on font design tokens.

```html
<h1 class="hero">Hero</h1>
<p class="title">title</p>
<p class="subtitle">subtitle</p>
<p class="caption">caption</p>
<p class="overline">overline</p>
```

> Keep in mind though, that the Website or Application should respect the [semantics of a HTML document](https://web.dev/learn/html/semantic-html/) to be compliant to web and A11Y standards.

<h1 className="sb-unstyled hero">Hero</h1>
<p className="sb-unstyled title">title</p>
<p className="sb-unstyled subtitle">subtitle</p>
<p className="sb-unstyled caption">caption</p>
<p className="sb-unstyled overline">overline</p>
