<Title>Menu</Title>

<Subtitle>
  A menu displays a list of choices on a surface that appears when interacting with an anchor element (e.g. a button). After choosing an option the menu ideally should be closed.
</Subtitle>

The `mw-menu` component can be used together with `mw-menu-item` as children to display a list of options.
The anchor element that should toggle the menu must be defined as `slot="anchor"` and is not limited to any specific element type.

<Canvas
  of={MwMenu.Default}
  withToolbar
  sourceState="shown"
  additionalActions={[
    {
      title: "Open in GitHub",
      onClick: () => {
        window.open("https://github.com/MaibornWolff/mwui/tree/main/mwui-stencil/src/components/mw-menu", "_blank");
      },
    },
  ]}
/>

> If it's preferred to use a standalone menu list that is not toggled or bound to an anchor, see [mw-menu-list docs](?path=/docs/components-menulist--default).

<ArgTypes of="mw-menu" />

## Placement

The `mw-menu` component uses `mw-popover` and therefore shares the placement options and described behavior infered from `popper.js`. See [mw-popover docs](?path=/docs/components-popover--default).

## Menu Items

A `mw-menu-item` can be provided with the following props:
