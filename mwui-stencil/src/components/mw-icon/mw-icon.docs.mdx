## Variable Font Style

As we use [Material Symbols](https://developers.google.com/fonts/docs/material_symbols), this icon font has variable styles that can be adjusted: **fill, weight, grade, and optical size**. The icon component leverages the usage and combines font-size and optical size. The **font-size**, however, can always be set individually using CSS.

> For most cases the default values provided by `mw-icon` should fit.

If there is a need to make use of the variable styles, just provide the props needed as described below. Find more information about the usage at Material Symbols guide: [font axes/variable styles](https://developers.google.com/fonts/docs/material_symbols).
