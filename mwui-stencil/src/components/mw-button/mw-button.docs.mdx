  Buttons execute an action, either in the background or foreground of an experience. Different button categories help guide users through certain actions. Buttons express what

`mw-button` is the primary component. It has 4 main variants:

- [Filled (default)](#filled)
- [Outline](#outline)
- [Ghost](#ghost)
- [Ghost Inverted](#ghost-inverted)

> The variants can be used in combination with the `icon` and `link` options.

### Icon

The button component can be used as an `IconButton` by omitting the label property and setting either a start- or end icon.

<Canvas of={MwButton.Icon} />

### With Icon

All variants can be extended with an icon at the start, end or both sides.
The `mw-button` component uses slots (`icon-start`, `icon-end`) to provide space for the icon.

> The order of the slots used does not matter.

```jsx
<mw-button test-id="button" label="BUTTON">
  <mw-icon slot="icon-start" icon="settings" />
  <mw-icon slot="icon-end" icon="settings" />
</mw-button>
```
