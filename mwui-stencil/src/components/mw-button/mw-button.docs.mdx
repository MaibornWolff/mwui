# Button

Buttons execute an action, either in the background or foreground of an experience. Different button categories help guide users through certain actions. Buttons express what action will occur when the user clicks or touches it either by text, icon, or both. Additional meaning can be derived from the button variant.

`mw-button` is the primary component. It has 3 main variants:

- [Filled (default)](#filled)
- [Outline](#outline)
- [Ghost](#ghost)

> The variants can be used in combination with the `icon` and `link` options.

### With Icon

All variants can be extended with an icon at the start, end or both sides.
The `mw-button` component uses slots (`icon-start`, `icon-end`) to provide space for the icon.

> The order of the slots used does not matter.

```jsx
<mw-button test-id="button" label="BUTTON">
  <mw-icon slot="icon-start" icon="settings" />
  <mw-icon slot="icon-end" icon="settings" />
</mw-button>
```
