# Autocomplete

The autocomplete allows the user to pick from a predetermined list of options, or enter their own text. The component allows either one or multiple entries if the `multiple` prop is set to true.

The dropdown options should be placed in the `dropdown-menu` slot. Additionally an icon can be added to the start of the input box using the `icon-start` slot.
