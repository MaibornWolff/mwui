# Dropdown

The dropdown component lets users pick from one of several choices by clicking on the input and selecting their choice from the menu. The options should be placed into the `dropdown-menu` slot as `mw-menu-item` components. Icons can be added to the component using the `icon-start` and `icon-end` slots.
